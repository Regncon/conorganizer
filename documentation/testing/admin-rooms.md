# Romadministrasjon i admin

Denne sjekklisten dekker `/admin/rooms` og `/admin/rooms/{pulje}`, der admin kan administrere rom og se romfordeling per pulje.

Denne filen er lagt til som del av dekningsinventaret. Sjekklisten skal fylles ut når vi tar romadministrasjon som egen fil i gjennomgangen.
